Some things you might be wondering about MCMap Live

How do I use colors from a texture pack?

  • Simply select "Try Colors..." in the Colors submenu and select the terrain.png from any texture pack.
  • You can also load up mcmap colors.txt files the same way.

Can I save these so they stay in the menu?

  • Yes! Add them to the list in "Preferences..."
  • You can add tweaked colors.txt files or texture pack PNGs directly.
  • Double click the name after adding to change it.

Can I make it so MCMap Live remembers the settings I like?

  • Set everything up just how you like it.
  • Open "Preferences..." and press "Set".
  • With no fanfare at all, your current settings are been memorized.
  • You can get rid of your settings at any time by clicking the "Reset" button.

How can I alter the performance of MCMap Live?

  • You may tweak the number of threads MCMap Live will use at once.
  • Open "Preferences..." and adjust the Rendering Threads setting.
  • An old machine or one with a slow hard drive will benefit from a setting of 2.
  • Most modern Macs see maximum speed with 4.
  • A Mac Pro or machine with an SSD can handle 8+

What do the keyboard keys do?

  • Cmd + / Cmd - : Zoom in and out in case you have no scroll wheel
  • Cmd 0 : Zoom to 100%
  • Up / Down arrows : Move the depth slider up or down one level
  • t / y : Adjust the horizontal tile spacing (hold shift for larger steps)
  • g / h : Adjust the vertical tile spacing (hold shift for larger steps)
  • r : Reset tile spacing

mcmap lets me export colors as a text file and edit them by hand. Can I do that here?

  1. Go to the Tools menu and select "Extract Colors from PNG..."
  2. An open file dialog will appear. Use this to select the terrain.png of the texture pack you want to use.
  3. If MCMap Live likes the file you chose, a save dialog will appear right away. Use this to save a text file containing all the color assignments.
  4. In the Options menu, select "Colors", then "Custom Colors..."
  5. An open file dialog will appear. Use this to select the text file you just saved.
  6. Your map will immediately update with the extracted colors!.
  7. Open the text file in a text editor.
  8. Locate the line with the block you want to change.
    • The first number on each line is a block ID. This will help you.
  9. The second, third, and fourth numbers are the RGB color value of the tile.
    • Make sure all the numbers are integers 0-255.
    • Use this to pick a color. You can read the RGB values on the bottom right.
  10. The fifth value is alpha. 0 is transparent, 255 is opaque.
  11. The sixth value is noise level. If you use the noise settings, larger integers are more noise. If you leave noise off, this number has no effect
    • (But it needs to be there even if it's zero!!)
